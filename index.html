Xuelei(Shelly) Peng's Portfolio Work Showcase

Accurate Chinese Text Image Generator(精准中文字生图)

Image generator that can generate accurate Chinese text, particularly useful in advertisement image generation, with higher text fidelity than all existing large models.

可以精准生成中文字的生图项目，特别适用于广告图的生成，中文字精度超过所有目前已有大模型。

Personalized Anime Character Generator(动漫角色生成器)

The first all-in-one-place anime character creator that allow users to create customized anime character with the consistent-looking, character's images, audioes and videoes. User can use natural language to characterize the companion and chat with the companion.

一个可以生成长相一致且有不同姿态、角色视频、音频等的动漫角色生成器，无需DreamBooth的训练即可快速生成多样的、长相一致的动漫角色。可通过文字、图片等生成角色的图片/视频。用户可以通过自然语言定制角色的性格，并与其对话。为国内外第一个一站式动漫角色/伴侣定制器。

2D Cartoon Mascot Generator(2D卡通吉祥物图像生成器)

a cartoon mascot/animal image generator, that can generate varied, consistent-looking cartoon animals, trained with kohya_ss and DreamBooth.

一个可以生成多样的、长相一致的卡通动物的图片生成器，基于kohya_ss和DreamBooth训练。

Virtual People Portrait Generator(虚拟人物肖像生成器)

a virtual people image generator, that generates consistent-looking, realistic people.

一个可以生成多样的、长相一致的虚拟数字人的肖像图片生成器。

2D Cartoon Image Generator(2D卡通图像生成器)

an anime style cartoon image generator.

一个可以生成动漫风格的卡通图片生成器。

AniSketch Sketch-to-Image Generator(草图转动漫精图生成器)

a web application that uses ControlNet to generate refined, anime-style images from scribbles. Obtained free GPU usage for hosting where 7-8% of applicants were granted free GPU.

使用了ControlNet将手绘草图生成为精细的动漫风格图片，支持草图的实时绘画。支持中英双语。此项目成功获得Hugging Face公司的免费GPU使用权，使用权获得成功率为7-8%。

Animefier Image-to-Image Generator(动漫化真人图片生成器)

a web application that uses ControlNet and Stable Diffusion models and Gradio that turns a realistic photo into one of an anime-style. Obtained free GPU usage for hosting from the hosting organisation where 7-8% of applicants were granted free GPU.

一个使用人工智能生成器将真实照片转变为动漫风格的图片的项目。支持中英双语。其质量与诸如Fotor等最佳商业替代品相媲美。此项目成功获得Hugging Face公司的免费GPU使用权，使用权获得成功率为7-8%。
