Micro-master Research Project - Train-free SOTA-level Identity Preservation in Image Generation(微硕士学位研究项目 - 无需训练的SOTA水平的图像生成中的身份保持)

Self-developed all technical contributions in an MPhil-level research project in image identity preservation. Supervised by: Prof. Jens Rittscher, Oxfor Univeristy. Benchmark results beats several top conference works(including those accepted by ICLR). Technicalities: UNet layer-specific feature blending, hierarchical blending routine, temporal-aware blending, layerwise-strength fade-out algorithm, feature-shift-combating mechanisms, reference feature leakage prevention, cross-attention-semantic-map assisted masking.

一门硕士级别暑期科研课程，旨在完成一篇关于图像生成中参考对象身份维护的研究论文，独立负责全部科研产出。导师：Prof. Jens Rittscher, 牛津大学，取得了同一研究问题中顶级质量的成果， 基准测试结果超越多项顶会成果（如ICLR等）。技术涵盖：UNet layer-specific feature blending, hierarchical blending routine, temporal-aware blending, layerwise-strength fade-out algorithm, feature-shift-combating mechanisms, reference feature leakage prevention, cross-attention-semantic-map assisted masking.
